# 과제(10) 순차리스트 응용 및 희소행렬 보고서

학과: 컴퓨터공학과 23학번  
이름: 김주혁  

## 1. 양의 정수 전용 순차 리스트 개선
- 기존 과제(9)의 realloc 기반 리스트를 확장하여 삽입, 삭제, 변경, 초기화 기능을 갖춘 메뉴형 프로그램으로 개선했습니다.
- 예외 처리 방어: 데이터 삽입 및 변경 시 item <= 0 조건을 통해 양의 정수만 입력되도록 방어했고, 유효하지 않은 인덱스 접근 시 시스템이 뻗지 않고 오류 메시지를 띄우도록 안정성을 높였습니다.

## 2. 희소행렬과 다중 포인터 구조 설계
- 조교님의 가산점 요구사항에 맞추어 SparseMatrix 구조체 내부에 순차 리스트를 가리키는 포인터(ArrayList *list)를 포함하는 다중 포인터 구조로 설계했습니다.
- 이를 통해 행렬의 크기 정보와 내부 데이터 리스트가 메모리상에서 독립적으로 유연하게 관리되도록 구현했습니다.
- 10x10 배열에서 추출한 20개의 원소를 전치(Transpose)시키는 과정에서, 조건문을 통해 전치행렬에 원소가 추가될 때마다 변수를 1씩 증가시켜 데이터 이동 횟수가 정확히 20회 발생함을 확인했습니다.
