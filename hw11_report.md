# 과제(11) 순차 리스트 기반 다항식 연산(곱셈) 보고서

**학과:** 컴퓨터공학과 23학번  
**이름:** 김주혁  

## 1. 다항식 데이터의 입력 및 파싱 로직
사용자가 입력한 `3X^2+2X^1+1X^0` 형식의 문자열을 순차 리스트에 저장하기 위해 C언어 표준 문자열 함수를 적극 활용했습니다.
- `strtok(buffer, "+")` 함수를 사용하여 '+'를 구분자로 삼아 각 항을 토큰(Token) 단위로 분리했습니다.
- 분리된 토큰(`3X^2`)은 `sscanf(token, "%dX^%d", &c, &e)`를 통해 계수(coef)와 지수(exp)로 안전하게 추출하여 구조체에 저장했습니다.

## 2. 수업시간 덧셈 코드의 오류 개선
기존 수업시간에 다룬 다항식 덧셈 로직은 두 배열의 인덱스를 비교하며 순차적으로 더해가는 방식이었습니다. 이 방식은 중간 차수가 비어있거나 입력이 정렬되지 않은 상태로 들어올 경우 인덱스 접근 오류나 계산 누락이 발생하기 쉽습니다. 
**[개선 방안]** 
이를 해결하기 위해 `insert_term`이라는 독립적인 삽입 모듈을 만들었습니다. 새로운 항이 들어오면 ① 기존 배열을 순회하며 동일한 차수가 있는지 검사하여 합산하고, ② 없을 경우 지수를 기준으로 내림차순 정렬될 위치를 찾아 삽입하는 방식입니다. 이를 통해 어떤 예외적인 다항식이 들어와도 안전하게 덧셈이 처리됩니다.

## 3. 다항식의 곱셈 구현
개선된 `insert_term` 함수를 활용하여 곱셈을 직관적으로 구현했습니다. 
- 이중 `for` 루프를 통해 다항식 A의 모든 항과 다항식 B의 모든 항을 교차합니다.
- 곱셈 공식에 따라 계수는 서로 곱하고(`A.coef * B.coef`), 지수는 더해준(`A.exp + B.exp`) 새로운 항을 `insert_term` 함수로 결과 리스트에 밀어 넣습니다.
- 동류항끼리의 합산과 차수 내림차순 정렬은 `insert_term` 함수가 내부적으로 모두 처리하므로, 코드의 중복을 줄이고 높은 신뢰성을 확보했습니다.
