# 과제(5) 배열 및 마방진 구현 보고서

**학과:** 컴퓨터공학과 23학번  
**이름:** 김주혁  

## 1. 정수 배열 실험
- `scanf`를 통해 배열 크기 `n`을 입력받고, `rand()` 함수로 난수를 생성하여 배열을 채웠습니다.
- 반복문을 이용해 순방향과 역방향 출력을 구현하였으며, `for`문 하나에서 총합과 최댓값, 최솟값을 동시에 계산하도록 효율성을 고려했습니다.

## 2. 홀수 마방진 구현
- 2차원 배열을 선언하여 1부터 $n^2$까지의 숫자를 배치했습니다.
- **배치 규칙:** 1. 첫 숫자는 첫 번째 줄 가운데에 배치합니다.
  2. 다음 숫자는 대각선 오른쪽 위 방향으로 이동하며 채웁니다.
  3. 만약 이동한 자리에 이미 숫자가 있거나 배열 범위를 벗어나면, 예외 처리를 통해 다음 위치를 계산했습니다.
- **검증 결과:** 3x3 입력 시 모든 가로, 세로, 대각선의 합이 15로 일치하는 것을 확인하였습니다.

## 3. 느낀 점
- 2차원 배열의 인덱스를 다루는 과정에서 범위를 벗어나는 오류를 해결하며 배열 구조를 더 잘 이해하게 되었습니다. 
- 조교님이 강조하신 대로 상수를 직접 쓰는 것보다 변수를 활용해 유연하게 코드를 짜는 연습을 했습니다.
