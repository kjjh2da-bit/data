# 과제(6) 포인터 및 동적 할당 결과 보고서

**학과:** 컴퓨터공학과 23학번  
**이름:** 김주혁  

## 1. 배열 알아보기 (동적 할당 적용)
- `scanf`를 통해 입력받은 정수 N의 크기만큼 `malloc` 함수를 사용하여 1차원 배열 공간을 메모리에 동적 할당하였습니다.
- 문제의 요구사항에 맞추어 인덱스 기호(`[]`)를 전혀 사용하지 않고, 오직 `*(arr + i)` 형태의 포인터 연산만을 활용해 난수 입력, 순방향/역방향 출력, 총합 및 최대/최소값 계산을 완벽하게 수행했습니다.
- 작업이 끝난 후에는 `free(arr)`를 호출하여 동적 할당된 메모리를 정상적으로 반환했습니다.

## 2. 홀수 마방진 (2차원 동적 할당 및 이중 포인터)
- 2차원 배열을 동적으로 할당하기 위해 먼저 int형 포인터의 배열을 `malloc`으로 생성하고, 반복문을 통해 각 행의 1차원 배열을 추가로 할당하는 방식을 취했습니다.
- 모든 배열 접근과 값의 수정은 `*(*(board + i) + j)` 형태의 이중 포인터 역참조 방식을 사용하여 구현했습니다.
- **검증 결과:** 3을 입력했을 때 가로 3세트, 세로 3세트, 대각선 2세트 등 총 8세트의 합이 모두 동일하게 출력되며 알고리즘이 정상 작동함을 확인하였습니다.
- 프로그램 종료 전 1차원 배열들을 먼저 `free()` 한 뒤, 마지막으로 포인터 배열을 `free()` 하여 메모리 누수(Memory Leak)를 방지하였습니다.

## 3. 고찰
기존의 직관적인 `arr[i]`나 `board[i][j]` 방식 대신 포인터의 덧셈 연산을 활용해 메모리 주소에 직접 접근하는 과정이 까다로웠지만, 이를 통해 메모리가 구조적으로 어떻게 연속 할당되는지 깊이 이해할 수 있었습니다.
