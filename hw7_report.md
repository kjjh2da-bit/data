# 과제(7) 구조체 배열 및 파일 입출력 결과 보고서

학과: 컴퓨터공학과 23학번  
이름: 김주혁  

## 1. 파일 입출력을 활용한 데이터 로드
- fopen과 fgets 함수를 사용하여 name_list.txt 파일에서 학생 이름 리스트를 읽어왔습니다. 
- 이때 텍스트 파일에서 한 줄을 읽어올 때 개행 문자(`\n`)가 포함되는 문제를 방지하기 위해 strcspn(file_buff, "\n")을 활용하여 문자열을 깔끔하게 처리한 뒤, 2차원 char 배열에 저장했습니다.

## 2. 배열 인덱스 연산자 활용
- malloc을 통해 입력받은 n 크기만큼의 학생 구조체(`Student`) 배열을 동적 할당했습니다.
- 난수 함수를 이용해 1~100 범위의 성적을 부여하고, 반복문을 돌며 직관적인 arr[i].score 방식으로 각 학생의 점수를 비교해 최고점과 최저점 인덱스를 산출했습니다.

## 3. 포인터 연산자 활용
- 두 번째 과제 요구사항에 맞추어 인덱스 기호(`[]`)를 일절 배제하고 구현했습니다.
- 구조체 포인터 접근 시 -> 화살표 연산자를 쓸 수도 있지만, 과제 명세에 명시된 *(arr + idx) 형태를 정확히 지키기 위해 (*(arr + i)).score` 형태로 멤버 변수에 접근하고 값을 수정했습니다.
- 두 연산 방식의 문법적 형태는 다르지만, 메모리상에서 구조체의 주소를 오프셋(offset)만큼 이동하여 접근한다는 본질적인 원리는 동일함을 출력 결과를 통해 완벽히 검증했습니다.
